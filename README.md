# unjumbler

unjumbler is a Clojure library that helps you solve [Jumble](http://en.wikipedia.org/wiki/Jumble) puzzles

## Usage

```clojure
(findwords "span")
=> ("span" "snap" "pans" "naps")

(findwordlist '("itnga" "kaowe" "nucpha" "otlbet"))
=> (("giant") ("awoke") ("paunch") ("bottle"))
```
